  • ResultPromise

    ResultPromise is a minimal Promise implementation that wraps a Result<T, E> instance. It supports the fulfil, await, map and flatMap combinators, facilitating a fluid sequencing of computations that produce either a Result (map) or a ResultPromise instance (flatMap). (The reject combinator is redundant, as failure is handled by the embedded Result.)
